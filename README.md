A simple analog clock to display current time, based on the host operating system. No internet connection required.

Controls:

1. Switch the black and white color:
  - Keyboard: Space
  - Mouse: Left-Click
  - Joypad: A, B, X, or Y
  - Touch Screen: Single Touch
   
2. Switch between windowed and fullscreen (Windows & Linux):
  - Keyboard: Tab
  - Mouse: Middle-Click
  - Joypad: Select
    
3. Exit the application:
  - Keyboard: Esc
  - Mouse: Right-Click
  - Joypad: Start
