Controls:
<br><br>
<ol>
  <li><b>Switch the black and white color:</b></li>
  <ul>
    <li>Keyboard: <i>Space</i></li>
    <li>Mouse: <i>Left-Click</i></li>
    <li>Joypad: <i>A, B, X, or Y</i></li>
    <li>Touch Screen: <i>Single Touch</i></li>
  </ul><br>
  <li><b>Switch between windowed and fullscreen (Windows & Linux):</b></li>
  <ul>
    <li>Keyboard: <i>Tab</i></li>
    <li>Mouse: <i>Middle-Click</i></li>
    <li>Joypad: <i>Select</i></li>
  </ul><br>
  <li><b>Exit the application:</b></li>
  <ul>
    <li>Keyboard: <i>Esc</i></li>
    <li>Mouse: <i>Right-Click</i></li>
    <li>Joypad: <i>Start</i></li>
    <li>Android: <i>Back Button</i></li>
  </ul>
</ol>
